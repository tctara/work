# CV (GitHub Pages)

Static, minimal CV site tailored to Chantara Tith with multiple role-focused versions.

## Files

- `index.html` — QA-focused CV version
- `software-engineer.html` — Software Engineer-focused CV version
- `technical-product-owner.html` — Technical Product Owner-focused CV version
- `event-operations.html` — Detailed page for event organization and coordination experience
- `projects.html` — Upcoming work and project roadmap showcase
- `styles.css` — shared minimalist responsive styling + light motion effects
- `resumes/` — downloadable PDF resume files for each role version
- `CV_INTAKE_TEMPLATE.md` — optional template for future CV version updates

## What's included

- Role-switch navigation between QA, Software Engineer, and Technical Product Owner pages
- Separate deep-dive page for event operations skills
- Projects page to showcase in-progress and upcoming work
- Resume download links on each role page
